Envio automático (requer API)

️ O envio 100% automático exige um serviço de API de WhatsApp (ex.: Z-API, Evolution API), geralmente pago. O app envia um POST em JSON no formato {phone, message} para a URL acima. Sem essa configuração, os lembretes aparecem na tela inicial prontos para enviar com 1 toque.

Como funciona: quando falta menos tempo que a antecedência escolhida, o lembrete aparece na tela inicial. Toque em "Enviar" e o WhatsApp abre com a mensagem pronta para o paciente.
